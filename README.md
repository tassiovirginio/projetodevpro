#PROJETODEVPRO

- Funciona nos Sitemas Operacionais
: Windows
: OSX
: Linux

- Requisitos
: Java 1.7
: Maven 3.x
: Git 2.x
: Banco de Dados (MySQL, Postgres, Derby entre outros)

- Instalarção/Run
: git clone git@bitbucket.org:tassiovirginio/projetodevpro.git
: cd projetodevpro
: mvn jetty:run

- Acesso após a instalação:
: http://127.0.0.1:9999
